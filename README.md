# CSCI-410

A repo containing the code/resources/frameworks used in making CSCI 410 (Introduction to datascience) final project.

**Language:** Python

**LLM Framework:** Matplotlib, Seaborn, Pandas, Numpy

**Database/Dataset:** Kaggle
